Hello Y

Y is a simple web application that allows you to create and share notes.

It is a work in progress.

It is currently being developed by frhd.me.

Features

  • Simple note creation
  • Simple note sharing
  • Simple note editing
  • Simple note deleting

We are working on adding more features. If you have any suggestions, please email us.

How to use

  • Click on the "Create Note" button to create a note.
  • Click on the "Share Note" button to share a note.
  • Click on the "Edit Note" button to edit a note.
  • Click on the "Delete Note" button to delete a note.
